GISVM Geoserver

Just another incredible idea by GISVM project
to bring you the power of Free GIS Server software.

This Virtual Machine is based on Free Software, built on Ubuntu Server version 16.04 LTS. Includes a selected set of free GIS Server software to provide you a complete virtual GIS server companion that you can run when and where you need the power of free GIS server software. It can be easily adapted to meet oneâ€™s personal needs, allowing the user to add new data or applications.

This Geoserver edition includes a powerful Webmap Server (Geoserver) to provide WMS and WFS map services from data that can be stored on a relational database server (PostgreSQL) with spacial support (PostGIS). Also includes several net services and configurations to make data communication between the virtual machine and the physical machine in such a way that allows GISVM to function as a simple machine within your computer network.

Included Tools

GISVM comes installed with some great full administration tools that you can use to completely manage it: Geoserver, Shell in a box and Webmin

Geoserver administration interface

http://gisvm:8080/geoserver
http://gisvm/geoserver
Login and configure GeoServer easily with your browser. Have total control over your Geoserver webmap server, add and configure web map services: WCS, WMS, WFS, etc.

Shell in a box

http://gisvm:4200
Get easy access to GISVM terminal in your browser. Have total control over your virtual machine from anywhere on your local network without the need of extra software.

Webmin

http://gisvm:10000
Webmin is a web-based interface for system administration for Unix. It gives you full remote control of GISVM Server, including the Ubuntu operating system, Samba windows sharing, Apache, SSH server and much more.

Webmin PostgreSQL

http://gisvm:10000/postgresql/
A simple web-based administration tool for PostgreSQL, included in Webmin. It is perfect for managing databases, browse tables, dump data, execute SQL commands, and much more.

Samba share: \\gisvm\gisdata

Samba integrates GISVM Server with any windows network and windows computer connected with your computer, including your own, if you are running on Windows operating system. It comes configured with a gisdata share directory that is accessible to windows at: \\gisvm\gisdata

Things You Can Do

GISVM is a virtual computer which runs within your physical computer. You can use GISVM server in many ways, as your personal local server, as a project team server available on your local network, or as a portable server that you can run on the road or at the office.

  • Easily adapt to your needs
  • Centralize your GIS data
  • Keep your GIS data organized
  • Run when you need it
  • Faster access to your GIS data
  • Easy share your GIS data
